# Global Superstore Power BI Dashboard

A professionally designed **Power BI dashboard** built using the widely recognized **Superstore dataset** to analyze sales performance, profitability, customer segments, and shipping behaviors. This project demonstrates strong skills in data modeling, DAX calculations, and dashboard UI/UX design.

---

## Overview

This dashboard provides comprehensive insights into Superstore performance across multiple dimensions. It helps answer key business questions such as:

- Which customer segments generate the most profit?
- How do sales and profit trends vary over time?
- Which product categories and regions drive the highest revenue?
- How does shipping mode impact order quantity?

## Dashboard Features

### **1. Profit Trend Analysis**
Shows year-to-year profit changes to identify growth patterns and business performance.

### **2. Region-wise Sales Map**
Geographical view of global sales performance.

### **3. Segment-wise Profit**
Comparison of Consumer, Corporate, and Home Office segments.

### **4. Ship Mode Distribution**
Pie chart visualizing order distribution across shipping modes.

### **5. Category-wise Revenue**
Breakdown of revenue generated by Technology, Office Supplies, and Furniture categories.
